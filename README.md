# performance-matrix

## Getting Started

To initialize your local project and push it to this GitHub repository, run the following commands in macOS Terminal:

### 1. Initialize Git Repository

```bash
git init
```

### 2. Stage All Files

```bash
git add .
```

### 3. Create the First Commit

```bash
git commit -m "Initial commit"
```

### 4. Add Remote Origin

```bash
git remote add origin https://github.com/hoosiersportsperformance-alt/performance-matrix.git
```

### 5. Push to GitHub

```bash
git push -u origin main
```

> **Note:** If your default branch is named `master` instead of `main`, replace `main` with `master` in the push command.
